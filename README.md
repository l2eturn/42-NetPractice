## Description
NetPractice is a general practical exercise designed to introduce the basics of computer networking. The goal of this project is to configure small-scale simulated networks to make them function properly. Throughout the 10 levels, this project covers core concepts such as configuring IP addresses, connecting devices through a router, and understanding the role of a default gateway within a network.

## Instructions
### How to run the training interface
1. Download the provided project file and extract it into a folder.
2. In that folder, run the `run.sh` script. This will launch a local web server and automatically open the training interface in your web browser.
3. Enter your intranet login in the interface to generate your personal configuration.

### Exporting Configurations & Submission Requirements
* Once you successfully complete a level, click the **"Get my config"** button to export and download your configuration.
* **Submission details:** You must complete all 10 levels. The 10 exported configuration files (one per level) must be placed directly at the root of your Git repository for submission.
